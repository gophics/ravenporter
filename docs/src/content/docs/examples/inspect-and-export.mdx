Use this when you are debugging an asset pipeline or handing data to another tool. It imports once, checks the report, runs `validate.Asset()` directly, and writes JSON IR.

## Key APIs

- `Result.Report`
- `validate.Asset()`
- `emit/json.WriteTo()`

## Run

```bash
go run ./examples/inspect-and-export
```

## Expected Output

```text
format=obj report_issues=0 validation_errors=0 validation_warnings=0 json_meshes=1 json_root_nodes=1
```
