Start here if you just want to see the library do something useful. This example writes a tiny OBJ file, imports it with `ImportPath()`, applies a built-in preset, and prints a few report fields.

## Key APIs

- `ravenporter.ImportPath()`
- `ravenporter.WithPreset()`
- `Result.Report`

## Run

```bash
go run ./examples/quickstart-import
```

## Expected Output

```text
format=obj preset=fast meshes=1 issues=0
```
