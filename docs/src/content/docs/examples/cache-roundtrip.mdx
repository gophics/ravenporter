This example covers the handoff from import time to runtime packaging. It imports a tiny asset, writes an `.rpcache` payload, opens it again, and closes the reopened package.

## Key APIs

- `cache.Write()`
- `cache.Read()`
- `(*cache.Asset).Close()`

## Run

```bash
go run ./examples/cache-roundtrip
```

## Expected Output

```text
source=obj cooked_meshes=1 summary_meshes=1
```
