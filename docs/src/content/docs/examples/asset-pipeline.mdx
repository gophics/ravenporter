If you want one example that ties the main library surfaces together, start here. It builds a small asset tree, loads a reusable profile, imports the batch, validates each result, exports JSON IR, and cooks a cache payload for every asset.

## Key APIs

- `ravenporter.ResolveProfile()`
- `ravenporter.SaveProfile()`
- `ravenporter.LoadProfile()`
- `ravenporter.ImportDir()`
- `validate.Asset()`
- `emit/json.WriteTo()`
- `cache.Write()`
- `cache.Read()`

## Run

```bash
go run ./examples/asset-pipeline
```

## Expected Output

```text
profile preset=quality scale=1.5 assets=2
  characters/hero.obj format=obj meshes=1 issues=0 validation_errors=0 validation_warnings=0 json_meshes=1 cache_meshes=1
  props/crate.obj format=obj meshes=1 issues=0 validation_errors=0 validation_warnings=0 json_meshes=1 cache_meshes=1
```
