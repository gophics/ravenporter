## Import A File

```go
package main

import (
	"context"
	"log"

	"github.com/gophics/ravenporter"
)

func main() {
	result, err := ravenporter.ImportPath(
		context.Background(),
		"assets/scene.glb",
		ravenporter.WithPreset(ravenporter.BuiltInPresetQuality),
	)
	if err != nil {
		log.Fatal(err)
	}

	log.Printf(
		"format=%s scenes=%d meshes=%d",
		result.Report.Source.DetectedFormat,
		len(result.Asset.Scenes),
		len(result.Asset.Meshes),
	)
}
```

## What You Get Back

`ImportPath()` returns `*ravenporter.Result`.

<Aside type="note" title="Check both return values">
  `err` tells you whether the import fully succeeded. If `result` is non-nil, keep the report anyway: it may still contain detected format, dependencies, or validation context that helps debugging.
</Aside>

| Field | Meaning |
| --- | --- |
| `Result.Asset` | The imported [`ir.Asset`](../ir-overview/) |
| `Result.Report.Source` | Input name, detected format, effective options, source metadata, and provenance notes |
| `Result.Report.Issues` | Stable issues grouped by pipeline stage and severity |
| `Result.Report.Dependencies` | Direct external dependencies discovered during import |
| `Result.Report.Summary` | High-level counts for scenes, meshes, materials, images, audio, fonts, and more |

## Choosing A Preset

RavenPorter ships three built-in preset names:

- `fast`
- `quality`
- `max-quality`

Use them through:

```go
ravenporter.WithPreset(ravenporter.BuiltInPresetFast)
ravenporter.WithPreset(ravenporter.BuiltInPresetQuality)
ravenporter.WithPreset(ravenporter.BuiltInPresetMaxQuality)
```

The preset sets the initial process-step mask. You can then layer explicit overrides on top of it with options such as `WithGlobalScale()`, `WithTargetUpAxis()`, `WithEmbedTextures()`, or a merged [`Profile`](../profiles-and-presets/).
