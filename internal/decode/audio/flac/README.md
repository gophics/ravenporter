# FLAC Decoder

## Format

Free Lossless Audio Codec.

## Extensions

`.flac`

## Supported Features

| Feature | Status |
|---------|:------:|
| Sample decoding (CONSTANT / VERBATIM / FIXED / LPC) | ✅ |
| Rice entropy coding (RICE / RICE2) | ✅ |
| Channel decorrelation (left-side / right-side / mid-side) | ✅ |
| 8-bit / 16-bit / 24-bit / 32-bit | ✅ |
| Mono / Stereo / Multichannel | ✅ |
| Duration | ✅ |
| Metadata (Vorbis comment) | ✅ |
| Embedded pictures (PICTURE block) | ✅ |
