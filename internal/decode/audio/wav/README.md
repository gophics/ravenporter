# WAV Decoder

## Format

RIFF/WAVE audio container.

## Extensions

`.wav`

## Supported Features

| Feature | Status |
|---------|:------:|
| PCM (integer) | ✅ |
| IEEE 754 float | ✅ |
| A-law (G.711) | ✅ |
| μ-law (G.711) | ✅ |
| WAVEFORMATEXTENSIBLE | ✅ |
| 8-bit | ✅ |
| 16-bit | ✅ |
| 24-bit | ✅ |
| 32-bit | ✅ |
| Mono / Stereo | ✅ |
| Sample decoding | ✅ |
| Duration | ✅ |
| MS-ADPCM compression | ✅ |
| IMA ADPCM (DVI) compression | ✅ |
| MP3-in-WAV (0x0055) compression | ✅ |
| Metadata tags (LIST/INFO) | ✅ |
| Loop points (smpl) | ✅ |
