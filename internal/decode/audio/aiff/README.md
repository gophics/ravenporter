# AIFF Decoder

## Format

Apple AIFF and AIFF-C (Audio Interchange File Format).

## Extensions

`.aiff`, `.aif`

## Supported Features

| Feature | Status |
|---------|:------:|
| AIFF (uncompressed BE) | ✅ |
| AIFF-C uncompressed (NONE) | ✅ |
| AIFF-C byte-swapped LE (sowt) | ✅ |
| AIFF-C 32-bit float (fl32) | ✅ |
| AIFF-C A-law (alaw) | ✅ |
| AIFF-C μ-law (ulaw) | ✅ |
| AIFF-C IMA ADPCM (ima4) | ✅ |
| 8-bit / 16-bit / 24-bit / 32-bit | ✅ |
| Mono / Stereo / Multichannel | ✅ |
| Sample decoding | ✅ |
| Duration | ✅ |
| Loop points (sustain + release) | ✅ |
| Metadata (NAME / AUTH / ANNO) | ✅ |
