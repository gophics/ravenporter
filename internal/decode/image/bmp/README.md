# BMP Decoder

High-performance decoder for Bitmap images.

## Extensions

`.bmp`

## Supported Features

| Feature | Status |
| :--- | :---: |
| 24-bit RGB | ✅ |
| 32-bit RGBA | ✅ |
| 4-bit / 8-bit Indexed Palette | ✅ |
| Bottom-Up / Top-Down Scanlines | ✅ |
| `BI_RGB` Uncompressed | ✅ |
| `BI_BITFIELDS` Masks | ✅ |
| RLE4 / RLE8 Compression | ✅ |
