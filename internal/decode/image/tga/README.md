# TGA Decoder

Truevision TGA raw fallback container module. Supports generic texture storage structures mapped back natively to uniform pipelines.

## Extensions

`.tga`

## Supported Features

| Feature | Status |
| :--- | :---: |
| 24/32 Bit BPR Resolving | ✅ |
| Scanline Vertical Flips | ✅ |
| Basic Uncompressed RGB | ✅ |
| Basic RLE Compression | ✅ |
| Index/Color-Mapping | ✅ |
