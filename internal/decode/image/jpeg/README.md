# JPEG Decoder

JPEG decoder using the Go standard library with custom ICC profile extraction.

## Extensions

`.jpeg`, `.jpg`

## Supported Features

| Feature | Status |
| :--- | :---: |
| Baseline & Progressive Decoding (stdlib) | ✅ |
| RGBA Pixel Extraction | ✅ |
| Channel Count Detection | ✅ |
| ICC Profile (APP2) Parsing | ✅ |
