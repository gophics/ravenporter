# PNG Decoder

PNG decoder using the Go standard library with custom APNG frame extraction.

## Extensions

`.png`

## Supported Features

| Feature | Status |
| :--- | :---: |
| Dimensions Pre-Scan (IHDR) | ✅ |
| RGBA Pixel Extraction (stdlib) | ✅ |
| Channel Count Detection | ✅ |
| APNG Frame Extraction (acTL/fcTL/fdAT) | ✅ |
