# DDS Decoder

DirectDraw Surface decoder providing lossless compressed passthrough to the Render Hardware Interface (RHI).

## Extensions

`.dds`

## Supported Features

| Feature | Status |
| :--- | :---: |
| DXT1 (`BC1`) | ✅ |
| DXT3 (`BC2`) | ✅ |
| DXT5 (`BC3`) | ✅ |
| ATI1 / BC4U (`BC4`) | ✅ |
| ATI2 / BC5U (`BC5`) | ✅ |
| Extended Header Support (`DX10`) | ✅ |
| DXGI BC6H (UF16/SF16) | ✅ |
| DXGI BC7 (UNORM/SRGB)| ✅ |
| Mipmap Count Extraction | ✅ |
| Uncompressed Formats | ✅ |
