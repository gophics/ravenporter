# PLY Decoder

Stanford PLY (Polygon File Format) decoder with ASCII and binary LE/BE support.

## Supported Features

| Feature | Status |
|---|---|
| ASCII format | ✅ |
| Binary little-endian | ✅ |
| Binary big-endian | ✅ |
| Vertex positions (x, y, z) | ✅ |
| Vertex normals (nx, ny, nz) | ✅ |
| Vertex colors (red, green, blue, alpha) | ✅ |
| Texture coordinates (s, t / texture_u, texture_v) | ✅ |
| Face indices (list property) | ✅ |
| Edge elements (vertex1, vertex2) | ✅ |
| N-gon fan triangulation | ✅ |
| Point clouds (vertices only, no faces) | ✅ |
| Header comments and obj_info | ✅ |
| char / int8 properties | ✅ |
| uchar / uint8 properties | ✅ |
| short / int16 properties | ✅ |
| ushort / uint16 properties | ✅ |
| int / int32 properties | ✅ |
| uint / uint32 properties | ✅ |
| float / float32 properties | ✅ |
| double / float64 properties | ✅ |
