# COLLADA Decoder

COLLADA 1.4.1 (`.dae`) scene file decoder.

## Supported Features

| Feature | Status |
|:---|:---:|
| Geometry (triangles + polylist + polygons + lines + tristrips + trifans + linestrips) | ✅ |
| Materials (Phong / Lambert / Blinn) | ✅ |
| Textures + images | ✅ |
| Visual scene hierarchy | ✅ |
| 4×4 matrix transforms | ✅ |
| Skeletal animations (T / R / S) | ✅ |
| Matrix animations (decomposed) | ✅ |
| Skinning + joint weights | ✅ |
| Morph targets | ✅ |
| Cameras (perspective + orthographic) | ✅ |
| Lights (point / directional / spot) | ✅ |
| Y_UP / Z_UP axis | ✅ |
| Multi-UV sets (set attribute) | ✅ |
