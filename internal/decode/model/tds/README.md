# 3DS Decoder

Autodesk 3D Studio (3DS) binary format decoder with full scene support.

## Supported Features

| Feature | Status |
|---|---|
| Geometry (vertices, faces, UVs) | ✅ |
| Smooth groups (stored for post-processing) | ✅ |
| Local transform matrix (3×4) | ✅ |
| Materials (diffuse, specular, ambient) | ✅ |
| Texture maps (diffuse, bump/normal) | ✅ |
| Point lights | ✅ |
| Spotlights | ✅ |
| Cameras (perspective) | ✅ |
| Keyframe animation (T/R/S tracks) | ✅ |
| Spline tension/continuity/bias | ✅ |
| Face material assignment | ✅ |
| Directional lights | ✅ |
| Hierarchical node tree | ✅ |
| Specular texture map (0xA204) | ✅ |
| Opacity texture map (0xA210) | ✅ |
| Reflection texture map (0xA220) | ✅ |
| Emissive texture map (0xA240) | ✅ |
