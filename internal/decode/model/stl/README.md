# STL Decoder

STereoLithography file format decoder with ASCII and binary support.

## Supported Features

| Feature | Status |
|---|---|
| Binary format | ✅ |
| ASCII format | ✅ |
| Facet normals | ✅ |
| Vertex positions | ✅ |
| Solid name parsing | ✅ |
| Per-facet color (RGB555 attribute byte) | ✅ |
| Magics-header part color (COLOR= tag) | ✅ |
