# Alembic Decoder

**Format:** Alembic (Ogawa backend)
**Extension:** `.abc`

## Supported Features

| Feature | Status |
|---|---|
| Ogawa binary container | ✅ |
| Mesh geometry (vertices, indices) | ✅ |
| Normals | ✅ |
| UVs (uv / st) | ✅ |
| Face triangulation (fan) | ✅ |
| Vertex colors (Cs property) | ✅ |
| Xform (4×4 matrix) | ✅ |
| Xform animation (multi-sample) | ✅ |
| Time sampling | ✅ |
| Material references (.arbGeomParams) | ✅ |
| Archive version / metadata | ✅ |
| Heuristic vertex scanning fallback | ✅ |
| Camera prims (ICamera schema) | ✅ |
| HDF5 backend (legacy, pre-2013) | ❌ |
